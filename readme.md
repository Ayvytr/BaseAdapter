# BaseAdapter

基于张鸿洋的BaseAdapter，最新改版后只支持androidx。废弃了Wrapper包装Adapter的做法，增加了`EmptyAdapter`，可继承实现空布局效果。请使用专业的下拉刷新库实现下拉刷新效果。





```
implementation 'com.ayvytr:base-adapter:1.4.1'
```
